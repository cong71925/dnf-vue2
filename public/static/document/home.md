# DNF团队数据管理系统
本系统主要面向于小团体，可以更方便地查看和管理每个群员的打团角色信息。
本项目可以登记角色，记录角色每次修改的数据，将其历史数据以图表显示。
对于奶系角色，还可以计算唱歌和偏爱后的buff数值和对C提升率。
在团队内还能通过表格或图表查看团员的角色的各项数据，以及伤害和buff的历史数据。
